# Resume Templates

Place your resume template files here. Each template needs a `.docx` file (the AI uses this as the base for tailoring).

## Recommended Templates

| File | Target Roles |
|------|-------------|
| `fullstack-ai.docx` | Full-stack, AI engineer, startup, LLM roles |
| `backend.docx` | Backend engineer, distributed systems, APIs |
| `platform-infra.docx` | Platform, infrastructure, DevOps, SRE |
| `ai-ml-infra.docx` | AI/ML infrastructure, MLOps, model serving |
| `general-swe.docx` | Generic SWE, unclear specialization |

You can start with just one general resume — the AI will tailor it per application.

Update `references/template-guide.md` if you add new templates.
